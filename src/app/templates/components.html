Predição de doenças cardíacas

Este projeto faz parte de uma atividade de cunho avaliativo para a disciplina de mineração de dados do curso Sistemas de Informação do IFNMG - Campus Januária.

O objetivo é escolher uma base de dados, realizar o seu pré-processamento e treinar esses dados com um algoritmo de Machine Learning. A base de dados escolhida refere-se a dados de uma pesquisa norte-americana que consiste em ligações para mais de 400 mil pessoas do país que são submetidas à um questionário sobre sua saúde. O intuito da pesquisa é fazer um levantamento se o entrevistado possui problemas cardíacos.

Os dados em questão foram devidamente pré-processados e aplicados em uma tarefa de classificação. A classificação consiste em trabalhar com os dados a fim de associar determinadas características à uma classe, ou seja, o objetivo é predizer a qual classe determinados dados pertencem. Para isso deve-se conhecer os dados previamente, definir uma classe alvo e treinar o algoritmo para que, ao atribuir dados novos, o sistema possa predizer corretamente a classe a qual os dados pertencem.

Utilizou-se mais de um algoritmo classificativo para a tarefa. Foram utlizadas métricas para obtenção do modelo mais eficiente para o contexto específico, além de todos os algoritmos terem passado pelo método de cross_validation.

Alerto que em hipótese alguma o resultado desta análise deve ser utilizada para atribuir qualquer diagnóstico. Se houver dúvidas quanto ao resultado, favor procurar um profissional da saúde.
